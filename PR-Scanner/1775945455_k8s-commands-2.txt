If we don’t want to install entire kubernates cluster, just need to install kubectl.exe to run basis kubectl command. 
Below is the process to install kubectl
Install kubectl binary with curl on Linux
Step-1  Download the latest release with the command:
curl -LO "https://dl.k8s.io/release/$(curl -L -s https://dl.k8s.io/release/stable.txt)/bin/linux/amd64/kubectl"

(FYI - to download a specific version use -  curl -LO https://dl.k8s.io/release/v1.26.0/bin/linux/amd64/kubectl)

Step-2 Validate the binary (optional) 
curl -LO "https://dl.k8s.io/$(curl -L -s https://dl.k8s.io/release/stable.txt)/bin/linux/amd64/kubectl.sha256"


Step-3 Validate the kubectl binary against the checksum file
       echo "$(cat kubectl.sha256)  kubectl" | sha256sum --check
(Note: Download the same version of the binary and checksum.)

Step-4 Install kubectl
        sudo install -o root -g root -m 0755 kubectl /usr/local/bin/kubectl

Step 5 Check installed version
       kubectl version --client --output=yaml
